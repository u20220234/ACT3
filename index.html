CONCEPTO DE CSS

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje de diseño utilizado para dar estilo y presentación a las páginas web. Con CSS, podés controlar colores, fuentes, tamaños, márgenes, posiciones y mucho más sin tocar la estructura del HTML.
